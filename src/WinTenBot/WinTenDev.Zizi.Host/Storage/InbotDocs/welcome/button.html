Welcome Button Tombol sebagai tambahan dari Welcome Teks. Beberapa tombol mungkin akan di tambahkan otomatis (di sebut tombol paten) dan tidak dapat di ubah tergantung pengaturan pada grub Anda. Akan tetapi Anda tetap dapat menambahkan tombol sesuai pengaturan /welcome. Adapun cara mengatur tombol adalah sebagai berikut. Pola: /welcome button Tombol 1 | http://example.com,.. /welcome button 📣 Channel|https://t.me/WinTenChannel, 🐐 Linux Community 🆔|https://t.me/LinuxGroupID, 📚 Bot Telegram API|https://t.me/TgBotID, 👥 Redmi 5A (Riva) ID|https://t.me/Redmi5AID, 🏠 HomePage|https://winten.tk Pisahkan tombol dengan tanda koma untuk menambahkan banyak tombol. Tidak perlu memikirkan pemetaan tombol, semua akan di buat dua kolom perbaris secara otomatis.
